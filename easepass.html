The web, navigable by hover alone.

Accessibility Surfer watches your cursor and clicks for you. Hover any link, button, input, or video thumbnail and after a few seconds it clicks itself. Built for anyone who can move a cursor with a mouse, head tracker, or eye gaze device but can't easily press a button. Works on every website out of the box.

Built for cursor-only users.

If you can move a cursor but can't easily press a button, Accessibility Surfer is for you. It works with any input that controls a pointer.

People with motor differences

Limited hand strength, tremors, RSI, fatigue, or anything else that makes the physical click difficult or painful.

Head tracker & eye gaze users

Assistive devices that move the cursor with a head, gaze, or other alternative input but don't always fire a real click on their own.

Classrooms & families

Drop-in Chrome extension. Install once, use on any laptop or Chromebook. No configuration to ship.

How it works.

1

Hover any clickable element

Links, buttons, inputs, video thumbnails, dropdowns. A blue ring appears centered on whatever your cursor is on.

2

Watch the countdown

The ring fills clockwise with a number in the center ticking down: 3 → 2 → 1. Default 3 seconds. Adjustable in the popup.

3

Accessibility Surfer clicks for you

At zero, the ring flashes green and a real click fires on the element. Links navigate, buttons activate, inputs focus.

4

Toggle anywhere, instantly

Tap Space, click the floating ✓/✕ button on any page, or open the toolbar popup. Off means off everywhere.

Built different by design.

✦

Universal coverage

Works on every site automatically. Links, buttons, inputs, dropdowns, even custom role="button" elements. No per-site setup.

▶

Smart YouTube mode

5-second dwell on thumbnails with the full ring, 3-second silent countdown on player and UI buttons in a status pill at the top.

Aa

Text accessibility built in

Change the font (OpenDyslexic, Lexend, Atkinson Hyperlegible, Arial, Comic Sans), size, line height, and letter and word spacing on any page from the popup.

↺

Resume on return

Move the cursor off and back on the same element within a second and the countdown picks up where it left off, so brief tremors or jitter don't reset progress.

How Accessibility Surfer compares.

OS-level dwell tools live in the system settings and treat the web as a generic surface. Accessibility Surfer lives in the browser and is tuned for it.

OS-level dwell tools

  • Buried in system accessibility settings
  • One dwell time for the whole computer
  • No awareness of video players, comments, or in-page UI
  • No font, size, or spacing controls for the page itself

Accessibility Surfer

  • Drop-in Chrome extension, on every site instantly
  • Separate dwell times for YouTube videos, YouTube buttons, and every other site
  • Smart YouTube mode with a top status pill and a per-thumbnail ring
  • Built-in text accessibility: font, size, line height, letter and word spacing

Works on every site.

Install once and Accessibility Surfer goes everywhere your browser does. Universal mode handles the long tail of the web with a 3-second dwell on any interactive element. Smart YouTube mode layers extra targeting on top for video thumbnails and player UI.

Smart mode

YouTube

5-second dwell ring on video thumbnails. 3-second silent countdown on player and UI buttons in a top status pill. Click anything, no mouse button required.

Universal mode

Every other site

Wikipedia, news articles, web apps, social media, banking, anywhere. A single 3-second dwell (adjustable) on every link, button, input, and dropdown.
